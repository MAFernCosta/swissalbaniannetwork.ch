# swissalbaniannetwork.ch

This project is a **React-based homepage** built using **Vite** as the package manager, styled with **TailwindCSS**, and featuring a responsive carousel powered by **react-multi-carousel**.  
The design is a **replica of an existing layout**, created as part of a **job interview assignment**.  
The site is deployed using **GitHub Pages**.

---

## **Tech Stack**
- **React** (UI framework)
- **Vite** (build tool)
- **TailwindCSS** (styling)
- **react-multi-carousel** (carousel component)
- **gh-pages** (deployment)
